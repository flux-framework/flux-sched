\doxysubsubsection{Detailed Description}
Expression evaluation API class. Parse and validate or evaluate an expression using the state of each individual predicates provided the target state of expr\+\_\+eval\+\_\+target\+\_\+t. Currently supported expression\+: An expression is one or more predicates combined with logical conjunction (\char`\"{}and\char`\"{}), disjunction (\char`\"{}or\char`\"{}), or parentheses (\char`\"{}()\char`\"{}). The evaluation order is consistent with these operators used in predicate calculus.
