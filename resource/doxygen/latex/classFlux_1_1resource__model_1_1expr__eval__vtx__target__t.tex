\doxysubsubsection{Detailed Description}
Expression evaluation resource vertex target class. Each resource vertex defines a set of predicates that return true or false based on its current state. This adaptor class enables us to evaluate those predicates without having to modify the resource vertex data types themselves. It is derived from \mbox{\hyperlink{classFlux_1_1resource__model_1_1expr__eval__target__base__t}{expr\+\_\+eval\+\_\+target\+\_\+base\+\_\+t}} and implements its validate and evaluate interfaces to do this work.
