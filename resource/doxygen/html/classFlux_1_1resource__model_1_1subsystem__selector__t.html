Detailed Description

template<typename graph_entity, typename inframap>
class Flux::resource_model::subsystem_selector_t< graph_entity, inframap >

For each chosen subsystem, a selector has std::set<string>. If edge/vertex's member_of[subsystem] matches with one of these strings, select that egde/vertex. Edge/vertex itself can also self-select. If its member_of[subsystem] has been annotated with '*', it is selected.
