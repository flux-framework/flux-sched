11 /*
12  * This jobspec module handles parsing the Flux jobspec format as specified
13  * in Spec 14 in the Flux RFC project: https://github.com/flux-framework/rfc
14  *
15  * The primary interface in the library is the Flux:Jobspec::Jobspec class.
16  * The constructor Flux::Jobspec::Jobspec() can handle jobspec data in a
17  * std::string, std::istream, or the top document YAML::Node as pre-processed
18  * by the yaml-cpp library.
19  *
20  * When errors are found in the jobspec stream the library will raise the
21  * Flux::Jobspec:parse_error exception. If the library was able to determine
22  * the location that the error occurred in jobspec yaml stream, it will appear
23  * in the position, line, and column members of the parse_error object. If it
24  * is unable to determine the location, all three of those fields will be -1.
25  *
26  * NOTE: The library will only be able to determine the location of error with
27  * yaml-cpp version 0.5.3 or newer.
28  */
