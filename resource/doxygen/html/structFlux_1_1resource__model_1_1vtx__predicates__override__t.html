Detailed Description

Assist expr_eval_vtx_target_t with an ability to override certain predicates associated with a resource vertex. Certain vertex predicates cannot be determined only by inspecting the vertex's own state, and this class helps override those predicates. Example: if the status of the parent resource vertex (e.g., compute node vertex) is "down", the status of a child vertex (e.g., compute core vertex) must also be "down" even if its status was not explicitly marked "down".
