Detailed Description

Expression evaluation resource vertex target class. Each resource vertex defines a set of predicates that return true or false based on its current state. This adaptor class enables us to evaluate those predicates without having to modify the resource vertex data types themselves. It is derived from expr_eval_target_base_t and implements its validate and evaluate interfaces to do this work.
